# Assets oficiales MaNGIA para deploy

Esta carpeta replica los auxiliares del flujo oficial dentro del paquete batch.

Por politica del repositorio, los FITS pesados no se versionan. Para correr la reconstruccion oficial completa en otra maquina, colocar aqui:

```text
MaStar_CB19.slog_1_5.fits.gz
```

Los archivos `.npy` pequenos de resolucion/longitud de onda pueden versionarse si estan presentes.
